## Problem: Aadhaar PDF Size Badi Kyun Hoti Hai?

Jab aap UIDAI website ya DigiLocker se Aadhaar card download karte hain, toh PDF ka size usually **1MB se 2MB** ke beech hota hai. Yeh isliye kyunki:

- Aadhaar PDF mein high-resolution image hoti hai
- Security features aur QR code embedded hota hai
- Digital signature bhi file size badha deta hai

Lekin zyada tar **government portals sirf 200KB-500KB** accept karte hain. Isliye compression zaruri hai.
