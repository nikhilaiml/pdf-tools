## PDF File Size Badh Kyun Jaati Hai?

PDF file ka size zyada hone ke kai karan hote hain:

- **High-resolution images** – Agar PDF mein photos hain toh size automatically badh jaata hai
- **Scanned documents** – Scanner se banayi gayi PDFs mein images as pages hoti hain
- **Embedded fonts** – PDF mein fonts embed hone se size increase hota hai
- **Multiple pages** – Zyada pages = zyada size
- **Unnecessary metadata** – Hidden data jo file mein stored rehta hai
