# DigiLocker PDF Download aur Use Kaise Kare – Complete Guide

DigiLocker bharat sarkar ka official document storage platform hai jahan aap Aadhaar card, PAN card, driving license, marksheets, aur bahut saare documents digitally store aur access kar sakte hain. Is guide mein hum batayenge ki **DigiLocker se PDF kaise download karein** aur unhe effectively use karein.

## DigiLocker Kya Hai?

DigiLocker ek free cloud storage platform hai jo Ministry of Electronics & IT (MeitY) ne banaya hai. Isme aap:

- Government-issued documents store kar sakte hain
- Original documents ki digital copies access kar sakte hain
- Sab documents ek jagah rakh sakte hain
- Documents share kar sakte hain securely

**DigiLocker pe available documents:**
- Aadhaar Card
- PAN Card (e-PAN)
- Driving License
- Vehicle Registration (RC)
- Class 10th / 12th Marksheets (CBSE)
- Insurance policies
- Property documents
- Aur bahut kuch...

## DigiLocker Se Document Download Kaise Karein

### Step 1: Login Karein

**App se:**
1. DigiLocker app download karein (Play Store / App Store)
2. Mobile number se register/login karein
3. Aadhaar se verify karein (optional but recommended)

**Website se:**
1. [digilocker.gov.in](https://digilocker.gov.in) kholein
2. "Sign In" pe click karein
3. Mobile number ya Aadhaar se login karein

### Step 2: Issued Documents Mein Jaayein

Login ke baad:
1. "Issued Documents" section pe jaayein
2. Yahan aapke sab government-linked documents dikhenge
3. Download karna chahte hain woh document select karein

### Step 3: PDF Download Karein

1. Document pe click karein
2. "Download" button pe click karein
3. PDF file aapke device mein save ho jaayegi

## DigiLocker PDFs Ko Process Kaise Karein

### Problem: File Size Badi Hai

DigiLocker se download ki gayi PDFs aksar 1-3MB ki hoti hain. Government portals sirf 200-500KB accept karte hain.

**Solution:** [Compress PDF](/tools/compress-pdf) tool se size kam karein.

### Problem: Multiple Documents Combine Karne Hain

Application forms aksar ek file mein sab documents maangte hain.

**Solution:** [Merge PDF](/tools/merge-pdf) se Aadhaar + PAN + Photo ek file banayein.

### Problem: JPG Format Chahiye

Kuch portals sirf image format accept karte hain.

**Solution:** [PDF to JPG](/tools/pdf-to-jpg) se convert karein.

### Problem: Sirf Ek Page Chahiye

Marksheet PDF mein sirf ek subject ka page chahiye?

**Solution:** [Split PDF](/tools/split-pdf) se specific pages extract karein.
