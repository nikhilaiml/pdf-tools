# Government Website PDF Upload Nahi Ho Raha? Yeh Karo

"File size exceeds limit" — yeh error message bahut common hai jab aap kisi government portal pe document upload karne jaate ho. Income Tax portal, EPFO, Passport Seva, scholarship portals — sab mein yeh problem aati hai.

Is guide mein hum aapko batayenge ki **government website pe PDF upload errors kaise fix karein** — step by step, bilkul free mein.

## Common PDF Upload Errors Government Portals Pe

Jab aap government website pe PDF upload karte hain, toh yeh errors aa sakte hain:

1. **"File size exceeds maximum limit"** — Most common error
2. **"Invalid file format"** — Wrong format (PDF chahiye lekin JPG upload ho raha)
3. **"File is corrupted"** — Damaged PDF file
4. **"Please upload a valid document"** — Generic error

### Sabse Common Reason: File Size Limit

**90% cases mein problem file size ki hoti hai.** Government portals ki size limits bahut strict hain:

| Portal | Max File Size | Accepted Formats |
|--------|-------------|------------------|
| Income Tax Portal | 1MB | PDF, JPG |
| EPFO / PF Portal | 500KB | PDF, JPG |
| Passport Seva | 200KB | JPG |
| IRCTC KYC | 5MB | PDF, JPG |
| Scholarship Portals | 500KB | PDF, JPG |
| State Govt Portals | 200KB - 300KB | PDF, JPG |
| Bank KYC | 200KB - 500KB | PDF, JPG |

## Portal-wise Specific Fixes

### Income Tax Portal (e-Filing)
- Max size: 1MB per document
- Accepted: PDF, JPG
- Fix: [Compress PDF](/tools/compress-pdf) se 1MB se neeche le aayein

### EPFO (PF Portal)
- Max size: 500KB
- Accepted: PDF
- Fix: Compress karke 400-500KB ke beech le aayein

### Passport Seva Kendra
- Max size: 200KB
- Accepted: JPG only
- Fix: [PDF to JPG](/tools/pdf-to-jpg) karein, phir JPG compress karein

### Scholarship Portals (NSP, State Scholarships)
- Max size: 500KB usally
- Accepted: PDF, JPG
- Fix: Document compress karein, multiple documents merge karein

## Pro Tips: Government Portal Upload Success

1. **Pehle size check karein** — Right-click → Properties se file size dekho
2. **Compression ke baad verify karein** — QR code, text, photo sab readable hona chahiye
3. **Browser change karein** — Chrome ya Firefox best hai government portals ke liye
4. **Cache clear karein** — Browser cache clear karke try karein
5. **Internet connection** — Stable connection use karein, WiFi better hai
6. **PDF file name** — Special characters (#, %, &) avoid karein file name mein
7. **Fresh download** — Aadhar/PAN dubara download karke try karein
