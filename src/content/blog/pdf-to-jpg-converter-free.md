## PDF vs JPG – Which Should You Use?

| Feature | PDF | JPG |
|---------|-----|-----|
| Best for | Documents, text | Photos, images |
| Text quality | Excellent (vector) | Lower (raster) |
| File size | Varies | Usually smaller |
| Editable | Yes (with tools) | No |
| Searchable text | Yes | No |
| Universal support | Very high | Very high |
| Social media | ❌ Not supported | ✅ Supported |
| Print quality | Excellent | Good |

**Rule of thumb:** Use PDF for documents you need to edit or print. Use JPG when you need images for sharing or uploading.
